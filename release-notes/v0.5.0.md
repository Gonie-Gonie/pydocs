# docscriptor v0.5.0

## Highlights

- Added fixed-layout `Sheet(...)` pages for short form-style documents such as certificates, awards, badges, and cover inserts.
- Added sheet items:
  - `TextBox(...)` for positioned text.
  - `Shape.rect(...)`, `Shape.ellipse(...)`, and `Shape.line(...)` for basic form decoration.
  - `ImageBox(...)` for positioned images such as logos and seals.
- Added explicit `z_index` ordering for sheet items.
- Added sheet background options, including renderer-safe solid colors and gradient-capable backgrounds.
- Sheets now behave as standalone pages by default:
  - DOCX uses section breaks to switch page size and margins around sheets.
  - PDF uses sheet-specific page templates so the actual PDF page media box can match the sheet size.
  - HTML renders sheets in a breakout page wrapper so wide sheets are not squeezed into prose layout.
- Updated the usage guide with a docscriptor-related contributor certificate example.

## Notes

- This is a minor release because it adds a new authoring model surface and renderer behavior for fixed-layout pages.
- Ordinary flowing documents continue to use the existing `Document`, `Chapter`, `Section`, `Paragraph`, `Table`, and `Figure` APIs.
